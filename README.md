# Offerte.app (Static + Vercel Functions + Gemini)

## Contenuto
- `index.html`: UI Offerte.app (ricerca live)
- `api/search.js`: Function Vercel che chiama Gemini con grounding web
- `vercel.json`: routing

## Deploy (GitHub -> Vercel)
1. Carica questa cartella su GitHub.
2. Importa il repo su Vercel.
3. Imposta su Vercel: Environment Variable
   - `GEMINI_API_KEY` = la tua API key Gemini
4. Deploy.
